# Testing framework for API and UI testing

## API Automation
### Stack used

* Rest-assured
* Java
* Junit 5

## UI Automation
### Stack used

* Selenide
* Java
* Junit 5

## How to run test

```
gradlew clean runApiTests runUiTests 
```
or run the script

```
sh api_ui_tests_interval_1d.sh
```
